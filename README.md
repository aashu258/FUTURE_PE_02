# FUTURE_PE_02
# AI-Powered UGC Ad Prompt System for MuscleBlaze Biozyme Performance Whey

## Project Overview

This project focuses on building a structured AI-powered prompt engineering system for generating high-converting UGC-style advertisements for MuscleBlaze Biozyme Performance Whey.

The main goal of this project is to demonstrate how AI prompt engineering can be used to create authentic, human-like, platform-ready fitness advertisements optimized for modern social media platforms such as Instagram Reels, YouTube Shorts, and Facebook Ads.

The generated advertisements are designed to feel natural, relatable, and realistic — similar to real gym-goers sharing personal experiences instead of traditional polished advertisements.

# AI Workflow

The workflow for this project was divided into two stages:

### Prompt Engineering & Structuring
The advanced prompts, frameworks, and prompt structures were designed using Claude AI.

### Content Generation & Refinement
The final ad scripts, hooks, tone adaptations, and platform-ready outputs were generated and refined using ChatGPT.

This workflow helped create highly structured prompts along with realistic and optimized marketing outputs.
